# Data sources & refresh

All per-country data is real and sourced; the pipelines are reproducible.

## Country profiles — `backend/data/country_profiles/*.json`

Generated by `backend/data/build_country_profiles.py`. The roster is every Ember country or
economy with a complete data year (demand + capacity + generation) and annual demand ≥ 1 TWh —
164 countries; micro-grids below that are excluded because a national merit-order model is not
meaningful there.

| Field | Source |
|-------|--------|
| Demand, installed capacity by tech, generation mix, annual capacity factors | **Ember Yearly Electricity Data** (CC-BY-4.0), latest *settled* year (provisional current-year releases excluded) |
| Delivered gas/coal price, discount rate (WACC), VRE capex multiplier | **Regional** levers from IEA WEO 2024 + IRENA 2024, keyed by `COUNTRY_REGION` |
| Heat rates, emission factors, dispatch/curtailment functions, storage, base capex/opex | Shared literature template (IEA WEO 2024 / IRENA 2024) |

Provenance (Ember source, settled year, per-country data year + region) is recorded in
`backend/data/country_profiles_manifest.json`.

Refresh:

```bash
make data          # rebuild from the cached Ember CSV
make data-refresh  # download the latest Ember release, then rebuild
```

The 49 MB Ember CSV is git-ignored (`backend/data/ember/`); `--download` fetches it. A test
(`test_profile_matches_ember_source`) independently re-derives every profile from the CSV and
fails if a shipped profile drifts from its cited Ember source.

## Hourly weather-year profiles — `backend/data/hourly/<CC>/<year>.csv.gz`

Generated by `backend/data/build_hourly_profiles.py`. Real hourly **solar** capacity factors from
**PVGIS** (JRC ERA5 reanalysis, 2017–2019), timezone-aligned to local noon and mean-scaled so each
country's annual solar CF matches Ember. Load and wind are the calibrated synthesizer for now
(real OPSD/EIA load+wind is the next increment). The app defaults to this `data` dispatch mode,
falling back to the synthesizer per country where no file exists.

```bash
make hourly        # rebuild from PVGIS (network, ~1 h for the full 164-country roster)
```
